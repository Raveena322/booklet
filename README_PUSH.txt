==========================================
  GITHUB PUSH - COPY & PASTE COMMANDS
==========================================

STEP 1: Create Repository
-------------------------
Go to: https://github.com/new
Create repository (don't add README)

STEP 2: Replace these values:
-------------------------
YOUR_USERNAME = your GitHub username
hd-booking-app = your repository name

STEP 3: Run these commands:
-------------------------

git remote add origin https://github.com/YOUR_USERNAME/hd-booking-app.git

git branch -M main

git push -u origin main

==========================================
That's it! Your code will be on GitHub!
==========================================
